# 方言マスター / Japanese Dialect Master

日本語学習者向けの日本各地の方言を学べるインタラクティブなウェブアプリケーションです。

## 🌟 特徴

- **インタラクティブな日本地図**: 四角形でデフォルメされた見やすい地図から都道府県を選択
- **詳細な方言分類**: 全47都道府県対応、主要都市の方言を網羅
- **豊富なフレーズ**: 各地域の代表的な方言フレーズと例文
- **フラッシュカード**: 効果的な記憶学習機能
- **クイズ機能**: 学んだ方言をテストして定着
- **進捗管理**: 学習の進み具合を自動保存
- **バイリンガル対応**: 日本語と英語の両言語をサポート
- **プロフェッショナルデザイン**: SVGアイコンと単色デザインで高級感のあるUI

## 🚀 セットアップ

### 前提条件

- Node.js (v16以上)
- npm または yarn

### インストール

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## 📱 使い方

1. **ホームページ**: 日本地図から学びたい地域を選択
2. **フレーズ一覧**: 選択した地域の方言フレーズを閲覧
3. **フラッシュカード**: カードをタップして方言を学習
4. **クイズ**: 学んだ内容をテストして確認

## 🎨 収録方言（主要）

### 北海道・東北
- 北海道弁
- 仙台弁（宮城弁）

### 関東・中部
- 名古屋弁（愛知弁）

### 関西
- **大阪弁** - ユーモアたっぷりのストレートな表現
- **京都弁** - 上品で婉曲的な表現
- **神戸弁（兵庫弁）** - 大阪弁と京都弁の中間
- **和歌山弁** - おおらかな響き

### 中国・四国
- 広島弁

### 九州・沖縄
- 博多弁（福岡弁）
- 沖縄弁（うちなーぐち）
